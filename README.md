# John H Hiemenbeger

## What is John H Hiemenbeger
John H Hiemenbeger is the best discord bot in the world. The aim of John H Hiemenbeger is to add at least one new feature every day.

## Current Commands:
* repeat (phrase) (times) - Repeat the same thing n times n <= 10
* dog - Show a random image of a dog using [https://random.dog]
* cat (basic:(extra information e.g., /orange, /say/Hello)) (advanced:(extra information e.g., filter=mono, fontSize=20)) - Show a random image of a cat using [https://cataas.com/cat]
* again - Repeat the last command
* quote - Say a random quote using [requests.get("https://dummyjson.com/docs/quotes")]
